Float

Alinear una imagen con un texto con float.

En el contenedor se usa:

.float {
margin-top: 20px;
background-color: gray;
border: 3px solid rgb(190, 127, 33);
overflow: hidden;
}

En la imagen:

.img {
max-width: 30%;
float: right;
margin: 40px 10px;
}
