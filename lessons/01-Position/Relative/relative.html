Position Relative

Usando Position Relative

Al usar (position: relative) permite que la caja se pueda mover usando: top, right, bottom, left y z-index.
El z-index influye en quién va a sobreponer entre las cajas.

Usando:

.box_1 {
background: lightgreen;
position: relative;
top: 10px;
left: 50px;
z-index: 2;
}

.box_2 {
background: lightsalmon;
position: relative;
z-index: 1;
top: -40px;
}

Manteniendo la posición de la caja en el flujo.

Usar position relative permite mover a cualquier lugar la caja si perder el espacio creado en el flujo.

Usando:

.box_3 {
background: grey;
position: relative;
top: 60px;
left: 90px;
}
