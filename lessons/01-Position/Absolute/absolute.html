Position Absolute

Usando Position Absolute

Al usar (position: absolute) hace que el elemento se coloque respecto a su contenedor más cercano. Si no ecnuentra, será el viewport.
No conserva su espacio en el flujo.

Usando:

.container {
border: 3px solid black;
width: 300px;
height: 300px;
margin-left: auto;
margin-right: auto;
/* al colocar position relative las cajas (box1,2,3...) será el que los contenga */
position: relative;
}

.box_1, .box_2, .box_3, .box_4, .box_5 {
width: 50px;
height: 50px;
}

.box_1 {
border: 4px solid tomato;
position: absolute;
left: 0;
top: 0;
}

.box_2 {
border: 4px solid rgb(43, 212, 27);
position: absolute;
right: 0;
top: 0;
}

.box_3 {
border: 4px solid rgb(145, 11, 66);
position: absolute;
left: 0;
bottom: 0;
}

.box_4 {
border: 4px solid rgb(141, 233, 22);
position: absolute;
right: 0;
bottom: 0;
}

.box_5 {
border: 4px solid rgb(63, 16, 139);
position: absolute;
top: 0;
right: 0;
left: 0;
bottom: 0;
margin: auto;
}
