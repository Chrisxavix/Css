Position Sticky

Usando Position Sticky

Es una mezcla entre relative y fixed.
Conserva el espacio en el Al hacer Scroll se moverá hasta lo establecido.

Usando:


.slide {
    width: 100%;
    background: gray;
    text-align: center;
    font-size: 38px;
    font-weight: bold;
    padding: 20px;
    /* Con line-height es como centrar o darle espacio entre la caja y el texto */
    line-height: 130px;
}

.menu {
    font-size: 32px;
    text-align: center;
    font-weight: bold;
    background: darkcyan;
    padding: 20px;
    width: 100%;
    position: sticky;
    /* Cuando se está en el top asigando se comporta como fixed, en 0 se ve bien xD */
    top: 20px;
}
