Listado con una imagen personalizada

Primero se resetea la clase contenedora

Usando:

.listImage {
list-style-type: none;
}

Darle estilos a los items de la clase contenedora

Usando:

.listImage__item {
background-image: url(./assets/check.png);
background-repeat: no-repeat;
background-position: 0 0;
background-size: contain;
display: block;
height: 20px;
padding-left: 25px;
margin-top: 20px;
}
