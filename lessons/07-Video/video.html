.video-responsive {
height: 0;
overflow: hidden;
padding-bottom: 56.25%;
padding-top: 30px;
position: relative;
}

Usando en el hijo del container:

.video-responsive iframe, .video-responsive object, .video-responsive embed {

position: absolute;
top: 0;
width: 100%;
}
