Order

Por defecto el ordenamiento lo trae en: order: 0;
Para cambiar el orden se entra en el item hijo, y se asigna el valor de 1 en adelante para posicionar: 


/* Contenedor Principal */
.grid__container {
    margin: 0 auto;
    width: 80vw;
    height: 100vh;
    background-color: #222;
    border: medium solid #000;
}

/* Contenedor de Orden */
.grid__order {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(2, 1fr);
}

/* El diseño para todos los items */
.grid__item {
    padding: 1%;
    font-size: 1.25rem;
    background-color: #666;
    border: 2px solid #999;
}

/* Dar el orden a items específicos */
.grid__item:nth-child(3) {
    background-color: rgb(128, 36, 36);
    order: 1;
}

.grid__item:nth-child(2) {
    background-color: rgb(36, 128, 100);
    order: 2;
}

Grid Align - justify-items

Observar los cambios desde "Inspeccionar Elementos"
Alinear de manera horizontal usando en la clase contenedora o padre:
justify-items: center;
Puede ser: justify-items: center; / start / end

Grid Align - Align items and Justify items


Observar los cambios desde "Inspeccionar Elementos"
Alinear de manera vertical y horizontal usando en la clase contenedora o padre:
align-items: center;
justify-items: center;
Se puede usar las propiedades de alineado de cada una de ellas 


/* Contenedor Principal */
.grid__container3 {
    margin: 0 auto;
    width: 80vw;
    height: 100vh;
    background-color: #222;
    border: medium solid #000;
}
    
/* Contenedor de alineación */
.grid__align3 {
    display: grid;
    grid-template-columns: repeat(3, 200px);
    grid-template-rows: repeat(2, 200px);
    align-items: center;
    justify-items: center;
}

/* El diseño para todos los items */
.grid__item3 {
    padding: 1%;
    font-size: 1.25rem;
    background-color: #666;
    border: 2px solid #999;

Grid Align - Alineando un hijo


Observar los cambios desde "Inspeccionar Elementos"
Se realiza con las mismas propiedades que el padre, pero en vez de (align/justify)-ITEMS, va (align/justify)-SELF. 


/* Contenedor Principal */
.grid__container4 {
    margin: 0 auto;
    width: 80vw;
    height: 100vh;
    background-color: #222;
    border: medium solid #000;
}
      
/* Contenedor de alineación */
.grid__align4 {
    display: grid;
    grid-template-columns: repeat(3, 200px);
    grid-template-rows: repeat(2, 200px);
}

/* El diseño para todos los items */
.grid__item4 {
    padding: 1%;
    font-size: 1.25rem;
    background-color: #666;
    border: 2px solid #999;
}

/* El diseño para el item personalizado */
.grid__item4:nth-child(5) {
    padding: 1%;
    font-size: 1.25rem;
    background-color: #666;
    border: 2px solid #999;
    align-self: end;
    justify-self: center;
}
