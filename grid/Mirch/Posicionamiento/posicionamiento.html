Posicionamiento

Para el posicionamiento se enfoca en la celda con:
.grid__item:nth-child(?) { }

Dentro de la clase del item seleccionado, se usa los valores por fila y columna:
Fila: Se le asigna un valor inicial y final por línea.
grid-row-start: ?;
grid-row-end: ?;

Para recotar código se le puede usar de esta manera:
grid-row: 2/4

Columan: la misma lógica que por filas:
grid-column-start: ?;
grid-column-end: ?;

Para recotar código se le puede usar de esta manera:
grid-column: 2/4

Ejemplo con la celda 7:

.grid__item:nth-child(7) {
color: red;
grid-column-start: 2;
grid-column-end: 4;
grid-row-start: 2;
grid-row-end: 4;
}
