Grid Areas

Se definen por secciones y con nombres que se asigna.

Declaradas con:
grid-template-areas:
"header header"
"content sidebar"
"footer footer";


Luego por cada sección, se le da el nombre a la clase declarada en el html, como:
item header,
item content,
item sidebar,
itemm footer

En la hoja de estilos iría:
.header { grid-area: header; }
.content { grid-area: content; }
.footer { grid-area: footer; }
.sidebar { grid-area: sidebar; }

Para dejar una area vacia se crea una nueva fila y se le asigna con un punto, entonces no se va a pintar la celda.
