Grid Explícita

Son las columnas y filas declaradas con valores, mientras que la Grid Implícita son los que no tienen valores asignados.

Para las separaciones entre celdas, sea mediante filas o columnas se usa:
column-gap: 12px
row-gap: 10px;

o simplicar estos dos usando:
gap: 10px 16px
Primero va row y luego column.

Usando:


.gridExplicit {
    display: grid;
    grid-template-columns: repeat(5, 1fr);
    grid-template-rows: repeat(4, auto);
    gap: 4px 4px;
}
