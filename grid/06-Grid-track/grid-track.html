Alinear de manera horizontal
Alinear la grilla dentro del contenedor padre con: justify-content: center; / start / end 


/* Contenedor Principal */        
.grid__container {
    margin: 0 auto;
    width: 80vw;
    height: 100vh;
    background-color: #222;
    border: medium solid #000;
    margin-top: 30px;
}

/* Contenedor de alineación */  
.grid__track {
    display: grid;
    grid-template-columns: repeat(3, 200px);
    grid-template-rows: repeat(2, 200px);
    justify-content: end;
}

/* El diseño para todos los items */  
.grid__item {
    padding: 1%;
    font-size: 1.25rem;
    background-color: #666;
    border: 2px solid #999;
}
